Overview

Purpose

The purpose of this site is to be able to find information about pokemon from various regions from the pokemon games. The user should be able to find information like name, types, and stats about each pokemon. They should be able to pull up information on the various pokemon one at a time.

Audience

The audience will be players of the pokemon games, specifically the older ones. Typically the age range will include a younger demographic, so the site should be simple and easy to use.
